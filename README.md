# Helixxed
A break-out clone where you need to mutate your ball with genes to achieve longer runs!
Made in Godot 4.6.1

## Playing the Game
Play the game on [itch.io](https://kaplin86.itch.io/helixxed)

## Core Gameplay
The main mechanic of this game are "Genes", which are power-ups you can equip on your ball to do special effects. These effects range from simple speed boosts, to things that wildly change how you play the game.
To make it past each round, you need to destroy all the bricks with a certain number of shots.

## How to build / modify
- Download the repository
- Import the project folder in Godot.
- No additional export settings are required.
- Use project > export in Godot to build.
